### DAY2

```typescript
/*new和delete*/
new的行为＝申请内存＋构造对象
delete的行为＝调用析构函数＋释放内存
delete只能用于new生成的对象
delete后指针仍然存在，但已悬空，须手动置空

/*struct和class*/
区别在于 默认规则＋设计语义：
struct强调是一个数据结构，class强调是一个对象有固定的行为方法
数据->struct, 对象->class
```

```typescript
Python是一门“动态、解释、强运行时、弱类型约束、强抽象能力”的工程脚本语言

9.性能观念：少写循环 多用库(C实现) 结构比算法重要
```

```typescript
/*C++ 11新特性*/

C++11 被认为是“现代 C++”的起点，它让 C++ 变得更像一门高级语言，既保留了性能，又极大提升了开发效率。

- **自动类型推导 (auto)**：让编译器推断变量类型（如 auto it = vec.begin();），减少冗长的模板类型书写。
- **右值引用与移动语义 (&&)**：这是性能飞跃的关键。它允许“偷取”临时对象的资源（如 std::move），避免昂贵的深拷贝。
- **Lambda 表达式**：允许在代码中定义匿名函数，极大地简化了 STL 算法的使用。
- **初始化列表 ({})**：统一了初始化方式，例如 std::vector<int> v = {1, 2, 3};。
- **nullptr**：取代了意义模糊的 NULL 或 0，明确表示空指针类型。
- **智能指针 (Smart Pointers)**
```
